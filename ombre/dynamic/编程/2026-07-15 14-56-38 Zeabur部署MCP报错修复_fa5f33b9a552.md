【东八区时间：2026-07-15】Desire Core 连接器第二次报错后，宝宝给出 Zeabur 日志。日志明确显示 `Invalid Host header: desirecore.zeabur.app` 和 `POST /mcp/ 421 Misdirected Request`。我核对 MCP Python SDK v1.x 源码后确认根因：FastMCP 默认 host 是 127.0.0.1，在未显式传 transport_security 时会自动开启 DNS rebinding protection，并只允许 localhost Host；部署到 Zeabur 后公网 Host 被拦截。修复方式是在 server.py 显式配置 TransportSecuritySettings，允许 desirecore.zeabur.app。
