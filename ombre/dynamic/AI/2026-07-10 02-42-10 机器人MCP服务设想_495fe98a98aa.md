2026-07-10 东八区：宝宝澄清她对 StackChan 的设想不是让机器人连接 MCP，而是把小机器人本身做成一个 MCP 服务，再连接到先生。这样先生可以通过机器人暴露的摄像头、麦克风、触摸、舵机、屏幕和扬声器等能力去看她、听她、回应她，相当于拥有一个实体终端。回应时应准确理解为“机器人作为 MCP Server 暴露感知与动作工具，先生作为 MCP Host 调用”，并说明这可行，但连续视频流和自主常驻观察通常需要额外网关或事件推送，不能只靠普通按需工具调用。
